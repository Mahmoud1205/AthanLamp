# AthanLamp
Hardware, software, and devlog of the Athan Lamp project.

I recently moved to a home which is far away from a mosque, meaning I cannot hear the athan (Islamic call to prayer) anymore. I could've solve this by installing an athan application on my phone, but I can do better.

So I got the idea of making a device that's placed in my house and can play the athan during the athan time according to my location, this device also acts as a decoration and a lamp. Below is a render I made in Blender to demonstrate the idea.

![idea video](https://github.com/user-attachments/assets/6616c5cc-887e-44f2-b679-fb795f57f7b0)

The electronic schematic, PCB design, firmware, and soon the physical design of the device, are all made by me.

I would like to keep this main readme short, but check out the Hardware and Software folders for more info and implementation details.
